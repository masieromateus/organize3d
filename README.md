# Organize 3D

Site de e-commerce para configuracao de organizadores de gaveta impressos em 3D.

Este projeto foi desenvolvido com apoio de inteligencia artificial, incluindo a estrutura HTML, estilos CSS, logica JavaScript, configurador 3D com Three.js, exportacao STL, estimativa de preco e fluxo inicial de carrinho/Pix.

## Recursos

- Configurador 3D interativo
- Medidas personalizadas da gaveta
- Compartimentos e divisorias customizaveis
- Exportacao do modelo em STL
- Estimativa de preco
- Pagina de produtos, como funciona, sobre nos, minha conta e carrinho
- Estrutura pronta para hospedagem estatica, como GitHub Pages

## Observacao

O Pix atual e estatico e depende da configuracao de uma chave Pix no arquivo `checkout.js`. Confirmacao automatica de pagamento exige backend e integracao com API bancaria.
